## 02. Solution

```Python
n, k = map(int, input().split())
student = [[0]*2 for _ in range(6)]

for _ in range(n):
    s, y = map(int, input().split())
    student[y-1][s-1] += 1

ans = 0
for a in range(6):
    for b in range(2):
        if(student[a][b]%k == 0):
            ans += student[a][b]/k
        else:
            ans += student[a][b]//k + 1
print(int(ans))
```
