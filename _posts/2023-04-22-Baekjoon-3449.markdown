## 02. Solution

### 01. Python (memory: 86116KB, time: 984ms)

```Python
for _ in range(int(input())):
    n1, n2 = input(), input()
    print(f"Hamming distance is {sum(n1[i] != n2[i] for i in range(len(n1)))}.")
```
