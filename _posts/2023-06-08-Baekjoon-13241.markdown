## 02. Solution

### 01. Python (memory: 31256KB, time: 128ms)

```Python
import sys
input = sys.stdin.readline

def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

a, b = map(int, input().split())
lcm = (a * b) // gcd(a, b)

print(lcm)
```
