# 👨‍💻 코드 <a name="코드"></a>

```cpp
float sinVal;
int toneVal;

void setup(){
  pinMode(8, OUTPUT);
}

void loop(){
  for (int x = 0; x < 180; x++){
  	
    // 사인파의 각도를 라디안 측도로 변환
    sinVal = (sin(x*(3.1412/180)));
    
    // 정현파에 의해 다른 주파수의 소리 생성
    value
    toneVal = 2000 + (int(sinVal*1000));
    tone(8, toneVal);
    delay(10);
  }
}
```
