## 02. Solution

```Python
import sys

heights = []
N = int(sys.stdin.readline())
n = current = before = 0

for i in range(N):
    heights.append(int(sys.stdin.readline()))

for _ in range(N):
    current = heights.pop()
    if current > before:
        n += 1
        before = current

print(n)
```
