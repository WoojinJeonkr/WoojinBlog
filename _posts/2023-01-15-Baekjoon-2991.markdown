## 02. Solution

```Python
A, B, C, D = map(int, input().split())
li = list(map(int, input().split()))
for n in li:
    attacked = 0
    if 0 < n % (A+B) <= A:
        attacked += 1
    if 0 < n % (C+D) <= C:
        attacked += 1
    print(attacked)
```
