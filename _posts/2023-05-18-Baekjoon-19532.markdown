## 02. Solution

### 01. Python (memory: 31256KB, time: 40ms)

```Python
a, b, c, d, e, f = map(int, input().split())
x = (c*e - b*f) // (a*e - b*d)
y = (c*d - a*f) // (b*d - a*e)
print(x, y)
```
