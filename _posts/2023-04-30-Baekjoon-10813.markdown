## 02. Solution

### 01. Python (memory: 31256KB, time: 48ms)

```Python
n, m = map(int, input().split())
basket = [str(i + 1) for i in range(n)]

for _ in range(m):
    i, j = map(int, input().split())
    basket[i - 1], basket[j - 1] = basket[j - 1], basket[i - 1]

print(" ".join(basket))
```
