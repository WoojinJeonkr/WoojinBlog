## 02. Solution

단순 구현 문제이다.<br>
while문을 통해 n이 0보다 크면 값을 받아 별표를 출력하고 n이 0이면 break를 통해 중단하도록 설계한다.

```Python
while True:
    n = int(input())
    if n > 0:
        for i in range(n):
            print('*'*(i+1))
    else: 
        break
```
