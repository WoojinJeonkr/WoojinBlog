## 01. Problem

[문제 확인하기](https://www.acmicpc.net/problem/2161){:target="_blank"}

## 02. Solution

```Python
n = int(input())
card = [i for i in range(1, n + 1)]
res = []

while True:
    res.append(card.pop(0))
    if not card:
        break
    card.append(card.pop(0))

print(*res)
```
