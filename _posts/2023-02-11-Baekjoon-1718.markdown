## 02. Solution

```Python
text, key = input(), input()

ans = ''

for i in range(len(text)):
    if text[i] == ' ':
        ans += ' '
    else:
        ans += chr((ord(text[i]) - ord(key[i%len(key)]) - 1) % 26 + ord('a'))

print(ans)
```
